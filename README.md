# ⚡ Learning KiCad: Hardware & PCB Design

This repository documents my transition from "scrap" electronics to professional **Monozukuri** (craftsmanship). It contains my schematics, board layouts, and project evolution as I master the art of PCB design for IoT and Audio applications.

---

## 🏆 Featured Project: 5V to 3.3V Linear Regulator
My first complete PCB design focused on stable power delivery for microcontrollers.

### ✅ Key Features & Skills Demonstrated:
- **Schematic Capture:** Proper use of decoupling capacitors and power symbols.
- **Physical Layout:** Managed footprints for SMD (AMS1117-3.3) and Through-Hole components.
- **Thermal Management:** Implemented **Via Stitching** to dissipate heat between copper layers.
- **Signal Integrity:** Designed **Copper Pours (Ground Zones)** on both layers to minimize electrical noise.
- **Design Rule Check (DRC):** Optimized the layout to achieve **Zero Violations** and **Zero Unconnected Items**.
- **Branding:** Custom silkscreen art and QR code integration for documentation.
